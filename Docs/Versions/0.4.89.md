# LOS 0.4.89

This update focuses on the timer-path proof after the 0.4.88 build completed but appeared to stall immediately after interrupts were enabled.

## Summary
- bumped the archive version to 0.4.89
- added an explicit visible first-timer proof line: `[OK] [Kernel] First timer IRQ observed.`
- changed the once-per-second timer heartbeat line to carry an `[OK]` tag so it is no longer suppressed by the current serial-line filter
- changed the idle task halt instruction to `sti; hlt` so the idle path always sleeps with interrupts enabled even if a saved flag state is stale
