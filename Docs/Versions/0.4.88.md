# LOS 0.4.88

This update repairs the shell-service link path introduced by the file split in 0.4.87 and also fixes a scheduler split regression that blocked the kernel build.

## Summary
- bumped the archive version to 0.4.88
- made the installed LOGIN/STRING image references weak so the first shell bootstrap link succeeds before disk-backed image objects are added
- added freestanding shell intrinsics for memcpy, memmove, and memset
- added a shell-local weak completion stub for user address-space finalization so the shell no longer hard-links against the memory-manager service image
- fixed the SchedulerDispatch split so its preprocessor guard no longer crosses file boundaries
- verified `Scripts/BuildBoot.sh` runs through the media EFI copy stage after these fixes
