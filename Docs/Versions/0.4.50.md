# Liberation OS 0.4.50

- Fixed the shell bootstrap build so shell sources are compiled with `LOS_EMBED_USER_IMAGE_BOOTSTRAP` during the first shell-service link pass.
- Reworked `ShellRuntime.c` so the bootstrap shell dispatches embedded `LOGIN.ELF` and `STRING.ELF` calls directly instead of requiring disk-backed installed-image symbols during the bootstrap link.
- Added a weak fallback for `LosMemoryManagerCompleteUserAddressSpaceCall` and replaced the remaining struct copy with explicit byte copying so the shell service no longer pulls in unresolved `memcpy` or memory-manager linker dependencies.
- Kept packaged build output out of the release archive.
