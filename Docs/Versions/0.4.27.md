# Liberation OS 0.4.27

- Added ELF64 program-header definitions for direct shell-side image execution.
- Implemented PT_LOAD segment materialization into a staged runtime image buffer.
- Added zero-fill handling for each loaded segment tail (`p_memsz - p_filesz`).
- Added direct entry invocation tracing before and after the loaded image call.
- Added a controlled stack-switching execution bridge for the disk-backed shell image path.
