# Liberation OS 0.4.100

- Disabled runtime function-entry tracing during the fatal x64 exception path so the fault report is no longer polluted or interrupted by nested `Enter ...` trace lines.
- Made the x64 interrupt diagnostics helpers non-tracing so they can print the requested fault state reliably: RIP, CS, RFLAGS, interrupted RSP, RBP, RAX..R15, CR2, and a 32-byte RIP hex dump.
- Kept the fault halt path unchanged after printing so the machine still stops immediately once the diagnostic report is emitted.
