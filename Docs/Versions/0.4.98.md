# Liberation OS 0.4.98

- Extended the x64 kernel fault diagnostics so the interrupt fault path now prints RIP, CS, RFLAGS, interrupted RSP, RBP, RAX through R15, and CR2.
- Added a 32-byte hex dump at RIP, using a current-CR3 page-table walk plus the higher-half direct map so the dump can report bytes without blindly dereferencing unmapped virtual addresses.
- The RIP-byte dump understands 4 KiB, 2 MiB, and 1 GiB mappings, which should make the first CPL3 failure far easier to localise.
