# LOS 0.4.84

- Reduced the default kernel-screen font scale and line spacing so the framebuffer console holds far more text before it needs to scroll.
- Added a simple non-reentrant screen-update guard so timer and spinner refreshes do not trample framebuffer log writes while the kernel is printing status lines.
- Rate-limited live timer overlay repainting, stopped the spinner once timer IRQs are known live, and refreshed the static header/footer after log-region scrolls to prevent the footer text from leaking into the log area.
