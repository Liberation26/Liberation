## LOS 0.4.3

This update continues the user-shell service conversion.

### Highlights
- adds a bootstrap transport model for the shell service using mailbox and channel records
- gives the shell service stable request and response endpoint identities
- routes init-shell bootstrap traffic through the shell service ABI, with loopback fallback while kernel transport is still being brought up
- keeps the shell installed as `/LIBERATION/SERVICES/SHELLX64.ELF`
- preserves the bootstrap launch path from InitCommand through CAPSMGR into SHELLX64
