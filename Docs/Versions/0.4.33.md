# Liberation OS 0.4.33

- Fixed ring-3 transition structure offsets used by the x64 entry trampoline.
- Fixed user call-block handoff so the mapped call-block user virtual address is passed in `rdi`.
- Added selector, RIP, RSP, and completion-address validation before user-mode entry.
- Added a kernel-owned completion return helper so user-mode completion status/result are staged consistently.
- Made the shell-side ring-3 path fail closed when the transition frame is invalid.
