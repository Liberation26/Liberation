# Liberation OS 0.4.35

- Unified shell command and library invocation through the same `LOS_USER_IMAGE_CALL` ABI.
- Added shell-owned completion/result staging buffers for all user-image calls.
- Added safe copy-back helpers so command/library status and result values are written back into shell space only after completion staging.
- Preserved disk-backed execution as the only normal command/library path.
