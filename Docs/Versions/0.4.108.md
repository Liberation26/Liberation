- 0.4.108: moved the X64 memory-manager handoff `Flags` field to the end of `LOS_X64_MEMORY_MANAGER_HANDOFF` so version 3 remains ABI-safe for existing memory-manager service binaries.
- The version constant remains `3`, but it now documents that version 3 means `Flags` was appended rather than inserted into the middle of the handoff layout.
- Kernel boot memory-map and handoff diagnostics now run after bootstrap-task registration instead of before memory-manager bootstrap setup, reducing the chance that verbose printing disturbs early scheduler timing.
